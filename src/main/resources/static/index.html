Book Inventory Backend API

This server only hosts the secured Spring Boot backend on localhost:8182.

Use the Angular frontend on localhost:4200 to log in and access the application screens.

Useful backend endpoints:

  • POST /api/v1/auth/login
  • GET /api/v1/auth/me
  • POST /api/v1/auth/logout
  • /swagger-ui.html

Opening API URLs directly in the browser is not the intended flow for this setup.
